Preface

About

E-Sports Management System (ESMS) is web application based on Laravel4 framework developed by ESMS team.

ESMS is application made for big gamming communities for on-line games. Application has built-in options to start and run tournaments and tracking all relevant data for community members.

System Requirements

E-Sports Management System (ESMS) is web application and for using it you need any device with internet connection.

Application is optimized for all devices and it has responsive design.

In case you are administrator and want to run this application on your server, you must meet the following:

  1. PHP version: 5.4 or above
  2. MySQL version: 5.6 or above
  3. Minimum of 100mb hosting space
  4. Apache mod_rewrite enabled
  5. PHP_fileinfo extension enabled
  6. MCrypt PHP extension enabled

Installation

If you are regular user, you do not have to install anything in order to use this application. All you have to do is to go on specified url: www.esms.com and login with your data

If you are administrator and you want to install applcation on your server, after meeting System Requirements you have just to extract esms.rar archive and upload it to your server.

After upload is done, you need to edit mysql.php file on line 3 to 6 with your database configuration. After that is done, go to your server URL, and run www.serverURL.com/public/mysql.php for ESMS to create required database and tables.

Getting started

Register account

To create new account, you have to navigate to top right corner, and in main menu click on "Register". After that, the following form will be displayed.

You must enter unique username, valid e-mail address and safe password. After you submit this form, we will send you e-mail message on the address you have provided, to activate your account.

Activate account

After submitting the register form, you will get e-mail with the activation link. Please click on that link and your account will be activated, and you will be able to access your player profile. Note that account has to be activated for user to access the application.

Login

To login with your account, you have to navigate to top right corner, and in main menu click on "Login". After that, the following form will be displayed.

Here you enter your account username and password. Account has to be activated.
